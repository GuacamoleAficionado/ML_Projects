Next, let's create a method to scale the features¶

Using the Z-score method:

$$ x_{j}^{(i)} = \frac{x_j^{(i)} - \mu_j}{\sigma_j} $$

We will use this function later to help make gradient descent run more predictably.

In [ ]:
def normalize_features(X):
    # calculate the mean and standard deviation by column in our matrix of features
    mu_j = np.mean(X, axis=0)
    sigma_j = np.std(X, axis=0)

    # get the array of normalized features
    X_norm = (X - mu_j) / sigma_j

    return X_norm

Homebrew Gradient Descent¶

In the implementation of gradient descent, it was helpful to break the algorithm up into several subroutines. Directly below we have done that, producing methods both to compute cost and all of it's partial derivatives.

Although not strictly necessary, vectorized code substantially improves the running time of gradient descent. This data set contains 506 rows of features and vectorizing the code as is done below takes the running time from over an hour down to just a few seconds.

The cost function $$J(θ) = \frac{1}{2m}\sum_{i=1}^m \left[ h_θ(\mathbf{\vec{x}^{(i)})-y^{(i)}} \right]^2 $$ may be vectorized like this:¶

$$ J(θ) = \frac{1}{2m}(Xθ-\vec{y})^T(Xθ-\vec{y}) $$¶

In [ ]:
def cost(X, y, theta, m, n):
    '''
    X - your normalized matrix containing m rows of n features including x_0 always set to 1
    y - your 1-D array containing m target variable instances
    theta - the 1-D array of weights including the bias (intercept)
    m - the number of oberservations (num rows in X)
    n - the number of features (num columns in X)
    '''
    # there are probably some redundant parentheses here
    return (1/2*m) * (np.transpose(X@theta - y)@(X@theta - y))

The derivative of the cost function can be vectorized this way $$ \left< \frac{\partial J(\theta)}{\partial\theta_1}, \frac{\partial J(\theta)}{\partial\theta_2}, ..., \frac{\partial J(\theta)}{\partial\theta_n}\right> = \frac{1}{m}X^T(X\theta - \vec{y}) $$¶

In [ ]:
def derivative(X, y, theta, m):
    return (1/m)*(X.T@(X@theta - y))

Putting It All Together¶

In [ ]:
def gradient_descent(X, y, m, n, k=10):
    '''
    a - the learning rate
    X - your normalized matrix containing m rows of n features including x_0 always set to 1
    y - your 1-D array containing m target variable instances
    theta - the 1-D array of weights including the bias (intercept)
    epsilon - if iter_diff < epsilon, then we terminate the loop
    iter - the current iteration number
    num_iters - the max number of iterations
    iter_diff - |previous_cost - current_cost| / current_cost
    m - the number of oberservations (num rows in X)
    n - the number of features (num columns in X)
    k - the interval at which you would like to keep a record of cost
    cost_history - keeps a record of the cost function output at each kth iteration of gradient descent
    '''
    a = 0.003
    theta = np.ones(n)
    epsilon = .00005
    num_iters = 100000
    iter = 0
    cost_history = []
    current_cost = 0
    iter_diff = 1

    while epsilon < iter_diff:
        if iter >= num_iters:
            break
        theta = theta - a*derivative(X, y, theta, m)
        previous_cost = current_cost
        current_cost = cost(X, y, theta, m, n)
        iter_diff = abs(current_cost-previous_cost)/current_cost
        if (iter%k == 0):
            cost_history.append(current_cost)
        iter = iter + 1
    return (theta, cost_history)

Now we're set up to train and test the model on each of the different folds.¶

In [ ]:
interval = 30  # interval is an optional argument for gradient_descent() that tells how often you want a snapshot of the process
predictions = [] # a list of arrays, each of which contains the predictions for one of the folds
fold_error = []  # stores the avg percent error for the predictions made on this fold

for fold in folds:

    train, test = fold[0], fold[1]
    
    # get subsets of training and testing data for current fold
    x_train = X[train]
    y_train = y[train]
    x_test = X[test]
    y_test = y[test]

    # the number of observations in this training set
    m = len(x_train)

    # normalize the features of the observations in the training/testing sets
    x_train = normalize_features(x_train)
    x_test = normalize_features(x_test)

    # adding θx=1 for x_0, θ_0
    x_train = np.c_[np.ones(m), x_train]
    x_test = np.c_[np.ones(len(x_test)), x_test]

    # the number of features in x_train
    n = x_train.shape[1]

    # optimize the weight vector by running gradient descent
    out = gradient_descent(x_train, y_train, m, n, interval)
    theta_hat, cost_history = out[0], out[1]

    # make predictions
    prediction = x_test.dot(theta_hat)
    predictions.append(prediction)
    err = sum(abs(prediction - y_test))/len(y_test)
    fold_error.append(err)

The most obvious naive method here would be to simply take the average value of MEDV (our target variable) and predict that average for each case. Well let's do just that and see what happens...
